1. WHAT IS NEXAGENT?
═══════════════════════════════════

NexAgent is a Telegram bot that gives you full remote control
over a Windows PC using natural language commands (Arabic or English).
It uses an AI provider (OpenAI GPT-4o or Google Gemini 1.5 Flash)
to understand your commands and execute them.

Key capabilities:
  - Instant screenshot
  - Open / close applications
  - File management
  - CPU / RAM / temperature monitoring
  - AI vision control (sees the screen and clicks)
  - Wake-on-LAN (remotely power on your PC)
  - Live screen streaming
  - Scheduled daily reports

═══════════════════════════════════
2. SYSTEM REQUIREMENTS
═══════════════════════════════════

  - Windows 10 or 11 (64-bit)
  - Python 3.11 or newer  <  python.org/downloads  >
  - Internet connection
  - API key: OpenAI (gpt-4o) or Google Gemini

═══════════════════════════════════
3. CREATE A TELEGRAM BOT
═══════════════════════════════════

  a) Open Telegram and search for @BotFather
  b) Send the command:  /newbot
  c) Choose a display name (e.g. My NexAgent Bot)
  d) Choose a username ending in "bot" (e.g. my_nexagent_bot)
  e) Save the "HTTP API token" BotFather gives you
     Example: 7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

═══════════════════════════════════
4. FIND YOUR CHAT ID
═══════════════════════════════════

  a) Open Telegram and search for @userinfobot
  b) Send any message to it
  c) It will reply with your info — save the "Id" number
     Example: 123456789

═══════════════════════════════════
5. INSTALLATION
═══════════════════════════════════

Method 1 — GUI installer (recommended):
  a) Double-click NexAgent_Setup.exe
  b) Follow the wizard steps
  c) Enter Bot Token and Chat ID when prompted
  d) Click Install

Method 2 — Run from source:
  a) Make sure Python 3.11+ is installed
  b) Open Command Prompt in the project folder
  c) Run:  build.bat
  d) Executable will be at dist\PCCommander\PCCommander.exe

═══════════════════════════════════
6. FIRST-TIME CONFIGURATION
═══════════════════════════════════

  a) Enter your Bot Token in the "Bot Token" field
  b) Enter your Chat ID in the "Allowed Users" field
     (multiple IDs separated by commas)
  c) Select your AI provider (OpenAI or Gemini)
  d) Enter the API key for your chosen provider:
     OpenAI:  platform.openai.com → API Keys → Create new key
     Gemini:  aistudio.google.com → Get API Key
  e) Click "Start Bot"

═══════════════════════════════════
7. FIRST RUN
═══════════════════════════════════

  a) Open Telegram and find your bot by its username
  b) Send:  /start
  c) A quick-command keyboard will appear
  d) Try: "take a screenshot" or tap the 📸 button

Example commands the bot understands:
  - "Take a screenshot"
  - "Open Chrome"
  - "What's the PC status?"
  - "Shut down in 30 minutes"
  - "List Desktop files"
  - "Start AnyDesk and send me the ID"
  - "Set volume to 50%"
  - "Lock the screen"

9. TROUBLESHOOTING
═══════════════════════════════════

Problem: Bot does not respond
  Fix: Verify the Bot Token is correct — restart the bot from Settings

Problem: "You are not authorized to use this bot"
  Fix: Make sure your Chat ID is listed in the Allowed Users field

Problem: AI key error
  Fix: Check your OpenAI or Gemini key — verify account has quota

Problem: Screenshots not sending
  Fix: Ensure Pillow and pyautogui are installed (re-run build.bat)

Problem: Bot stops after a while
  Fix: Enable "Watchdog" in settings — it will restart the bot on crashes

Problem: vision_* commands not working
  Fix: These require OpenAI GPT-4o only — they do not work with Gemini

Problem: Voice messages not transcribed
  Fix: Use the OpenAI provider — Gemini does not support Whisper transcription

Problem: Bot forgets conversation history
  Fix: Normal after 2 hours of inactivity — contexts are cleaned automatically

Problem: Build error "Python not found"
  Fix: Install Python and check "Add Python to PATH" during setup

═══════════════════════════════════
10. SECURITY NOTES
═══════════════════════════════════

  - Never share config.json — it contains encrypted keys
  - Never delete secret.key — without it you cannot decrypt your settings
  - Enable PIN in settings for an extra security layer
  - Read DOCS.md for the full multi-layer security model details
  - Only add Chat IDs of people you fully trust — they get complete PC control
