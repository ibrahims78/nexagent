═══════════════════════════════════
1. نبذة عن البرنامج
═══════════════════════════════════

NexAgent هو بوت تيليغرام يتيح لك التحكم الكامل بحاسبك الشخصي
(ويندوز) عن بُعد عبر أوامر طبيعية بالعربية أو الإنجليزية.
يستخدم الذكاء الاصطناعي (GPT-4o أو Gemini 1.5 Flash) لفهم أوامرك وتنفيذها.

الإمكانيات الرئيسية:
  • لقطة شاشة فورية
  • فتح/إغلاق التطبيقات
  • إدارة الملفات
  • مراقبة CPU / RAM / درجة الحرارة
  • التحكم البصري الذكي (يرى الشاشة وينقر)
  • إيقاظ الحاسب عن بُعد (Wake-on-LAN)
  • بث الشاشة مباشرة
  • تقارير يومية تلقائية

═══════════════════════════════════
2. متطلبات التشغيل
═══════════════════════════════════

  - ويندوز 10 أو 11 (64-bit)
  - Python 3.11 أو أحدث  <  python.org/downloads  >
  - اتصال بالإنترنت (لتشغيل البوت)
  - مفتاح API: OpenAI (gpt-4o) أو Google Gemini

═══════════════════════════════════
3. إنشاء بوت تيليغرام
═══════════════════════════════════

  أ) افتح تيليغرام وابحث عن @BotFather
  ب) أرسل الأمر:  /newbot
  ج) اختر اسماً للبوت (مثال: My NexAgent Bot)
  د) اختر اسم مستخدم ينتهي بـ "bot" (مثال: my_nexagent_bot)
  هـ) احفظ رمز "HTTP API token" الذي سيُعطيك إياه BotFather
      مثال: 7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

═══════════════════════════════════
4. معرفة Chat ID الخاص بك
═══════════════════════════════════

  أ) افتح تيليغرام وابحث عن @userinfobot
  ب) أرسل أي رسالة له
  ج) سيردّ بمعلوماتك — احفظ الـ "Id" (رقم مثل: 123456789)

═══════════════════════════════════
5. تثبيت البرنامج
═══════════════════════════════════

الطريقة الأولى — التثبيت التلقائي (مُوصى به):
  أ) انقر نقراً مزدوجاً على NexAgent_Setup.exe
  ب) اتبع خطوات المعالج
  ج) أدخل Bot Token و Chat ID عند الطلب
  د) انقر "تثبيت"

الطريقة الثانية — التشغيل من المصدر:
  أ) تأكد من تثبيت Python 3.11+
  ب) افتح Command Prompt في مجلد المشروع
  ج) شغّل: build.bat
  د) الملف التنفيذي سيكون في dist\PCCommander\PCCommander.exe

═══════════════════════════════════
6. إعداد البرنامج لأول مرة
═══════════════════════════════════

  أ) أدخل Bot Token في حقل "رمز البوت"
  ب) أدخل Chat ID في حقل "المستخدمون المسموح لهم"
     (يمكن إدخال أكثر من معرّف مفصولاً بفواصل)
  ج) اختر مزود الذكاء الاصطناعي (OpenAI أو Gemini)
  د) أدخل مفتاح API الخاص بالمزود المختار
     OpenAI:  platform.openai.com → API Keys → Create new key
     Gemini:  aistudio.google.com → Get API Key
  هـ) انقر "تشغيل البوت"

═══════════════════════════════════
7. التشغيل الأول
═══════════════════════════════════

  أ) افتح تيليغرام وابحث عن بوتك باسم المستخدم
  ب) أرسل:  /start
  ج) ستظهر قائمة الأزرار السريعة
  د) جرّب: "خذ لقطة شاشة" أو انقر على زر 📸

أمثلة على أوامر يفهمها البوت:
  • "خذ لقطة شاشة"
  • "افتح المتصفح"
  • "أرسلي حالة الحاسب"
  • "اغلق الحاسب بعد 30 دقيقة"
  • "اعرض ملفات سطح المكتب"
  • "شغّل AnyDesk وأرسل الـ ID"
  • "ضبط الصوت على 50%"
  • "اقفل الشاشة"

═══════════════════════════════════
8. جدول الملفات المهمة
═══════════════════════════════════

  الملف/المجلد                          الوصف
  ─────────────────────────────────────────────────────────
  PCCommander.exe                        الملف التنفيذي الرئيسي
  %APPDATA%\PCCommander\                 مجلد البيانات
  %APPDATA%\PCCommander\config.json      إعدادات البرنامج (مشفرة)
  %APPDATA%\PCCommander\secret.key       مفتاح التشفير (لا تحذفه!)
  %APPDATA%\PCCommander\logs\            ملفات السجلات
  %APPDATA%\PCCommander\downloads\       الملفات المستلمة من تيليغرام
  %APPDATA%\PCCommander\screenshots\     لقطات الشاشة المحفوظة

═══════════════════════════════════
9. حل المشكلات الشائعة
═══════════════════════════════════

المشكلة: البوت لا يستجيب
  الحل: تأكد من صحة Bot Token — أعِد تشغيل البوت من نافذة الإعدادات

المشكلة: "غير مصرح لك باستخدام هذا البوت"
  الحل: تأكد أن Chat ID مدرج في قائمة "المستخدمين المسموح لهم"

المشكلة: خطأ في مفتاح AI
  الحل: تحقق من مفتاح OpenAI أو Gemini — تأكد من وجود رصيد أو استخدام الخطة المجانية

المشكلة: لقطة الشاشة لا تُرسل
  الحل: تأكد أن Pillow و pyautogui مثبتان (شغّل build.bat مجدداً)

المشكلة: البوت يتوقف بعد فترة
  الحل: فعّل "Watchdog" من الإعدادات — سيُعيد تشغيل البوت تلقائياً عند الانهيار

المشكلة: أوامر vision_* (التحكم البصري) لا تعمل
  الحل: هذه الأوامر تتطلب OpenAI GPT-4o فقط — لا تعمل مع Gemini

المشكلة: الرسائل الصوتية لا تُحوَّل
  الحل: استخدم مزود OpenAI — Gemini لا يدعم تحويل الصوت عبر Whisper

المشكلة: البوت لا يتذكر المحادثة السابقة
  الحل: طبيعي بعد ساعتين من الخمول — تُحذف السياقات تلقائياً لتوفير الذاكرة

المشكلة: خطأ عند البناء "Python not found"
  الحل: ثبّت Python وتأكد من إضافته لمتغير PATH أثناء التثبيت

═══════════════════════════════════
10. ملاحظات الأمان
═══════════════════════════════════

  ✓ لا تشارك ملف config.json مع أحد — يحتوي على مفاتيح مشفرة
  ✓ لا تحذف secret.key — بدونه لن تتمكن من فك تشفير إعداداتك
  ✓ فعّل رمز PIN من الإعدادات لطبقة أمان إضافية
  ✓ راجع DOCS.md للتفاصيل الكاملة عن نموذج الأمان متعدد الطبقات
  ✓ لا تُضف Chat IDs لأشخاص لا تثق بهم تماماً — لديهم تحكم كامل بالحاسب
