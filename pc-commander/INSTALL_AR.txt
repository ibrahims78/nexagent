═══════════════════════════════════════════
 المتطلبات الأساسية قبل البناء
═══════════════════════════════════════════

1. Python 3.10 أو أحدث
   → تحميل: https://www.python.org/downloads/
   → تأكد من تفعيل "Add Python to PATH" أثناء التثبيت

2. Inno Setup 6 (لإنشاء setup.exe)
   → تحميل: https://jrsoftware.org/isdl.php
   (اختياري - يمكن تشغيل البرنامج بدونه)

═══════════════════════════════════════════
 خطوات البناء
═══════════════════════════════════════════

1. افتح Command Prompt في مجلد المشروع
2. شغّل: build.bat
3. انتظر اكتمال البناء (5-10 دقائق)
4. ستجد الملفات في مجلد dist\

═══════════════════════════════════════════
 إعداد البرنامج بعد التثبيت
═══════════════════════════════════════════

── تيليغرام ──────────────────────────────
1. افتح تيليغرام وابحث عن @BotFather
2. أرسل: /newbot
3. اختر اسماً للبوت
4. احفظ الـ Token الذي سيُعطيك إياه

للحصول على User ID الخاص بك:
→ ابحث عن @userinfobot في تيليغرام وأرسل له /start

── الذكاء الاصطناعي ──────────────────────
OpenAI:
→ platform.openai.com → API Keys → Create new key

Google Gemini (مجاني):
→ makersuite.google.com → Get API Key

── الاتصال (Cloudflare - موصى به) ────────
Cloudflare Tunnel مجاني ولا يحتاج حساب!
سيتم تحميل cloudflared تلقائياً.

── الاتصال (ngrok - بديل) ─────────────────
→ إنشاء حساب مجاني: dashboard.ngrok.com
→ Your Authtoken → نسخ الـ Token
